Selector: markdown-editor

@Input() placeholder: string

The placeholder value for the editor textarea element.

@Input() font: string;

The preferred editor font. For available values, refer to the fonts array in markdown-toolbar.component.ts. Default = Consolas.

@Input() fontSize: number;

The preferred editor font size. Default = 16px.

@Input() minFontSize: number;

Sets the minimum range of font sizes when generating the available range of configurable font sizes in the editor toolbar. Default = 10.

@Input() maxFontSize: number;

Sets the maximum range of font sizes when generating the available range of configurable font sizes in the editor toolbar. Default = 20.

@Input() showToolbar: boolean;

Determines whether the toolbar should be shown.

@Input() minRows: number;

Sets the matAutosizeMinRows value for the textarea element. Default = 20.

@Input() maxRows: number;

Sets the matAutosizeMaxRows value for the textarea element. Default = 25.

@Input() markdown: MarkdownService;

Specifies the instance of MarkdownService the editor will use.
